Frequently Asked Questions

How does Monkey Block work?

Monkey Block tracks your time on distracting websites and shows a visual countdown timer that changes color as time runs out. When you reach your daily limit, the site gets blocked with a motivational message until the next day.

Does the timer appear on every website?

No, the visual timer only appears on websites you've added to your limits or blacklist. It won't show on regular browsing or work-related sites unless you specifically add them.

Can I customize which sites to block?

Absolutely! You can add any website to your blacklist, set individual time limits for each site, or use combined limits for all sites together.

Does Monkey Block work in incognito mode?

No, extensions need to be explicitly enabled for incognito mode. Go to Chrome Extensions settings and enable "Allow in incognito" if you want to use Monkey Block in private browsing.

Is it really free?

Yes! The core features are free to use. We're planning premium features for power users in the future, but essential blocking and timer functionality will always remain free.
